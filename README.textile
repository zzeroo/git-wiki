h1. git-wiki

A wiki engine that uses a Git repository as its data store.

h2. Status

Under heavy development at the moment.

h2. Requirements

* rubygems
* sinatra
* grit
* redcloth
* rubypants

You can install these requirements via bundler. Just install bundler: gem install bundler.
Then 'bundle install' installs all the gem.

h2. Configuration

Setup the root of your wiki (where the wiki pages and the git repo lives) in @environment.rb@
